# Военно-космическая академия им. А.Ф. Можайского — лендинг

Лендинг и бэкенд для проекта «МАФ» (Можайский Александр Фёдорович).

## Стек

- **Бэкенд:** Go (стандартная библиотека `net/http`, встроенная раздача статики)
- **Фронтенд:** HTML, CSS, JavaScript (статичные файлы в `cmd/server/static/`)

## Запуск

Из корня проекта:

```bash
go run ./cmd/server
```

Сервер слушает порт **8080**. Открой в браузере: http://localhost:8080

```
vka_bot_ai/
```

## API

- `GET /` — главная страница (лендинг)
- `GET /api/health` — проверка работы сервера (`{"status":"ok"}`)
- `POST /api/chat` — чат с ChatGPT. Тело: `{"message": "текст"}`. Ответ: `{"reply": "..."}`.

## Интеграция ChatGPT и кодовое слово «маф»

Для работы чата задай переменную окружения **OPENAI_API_KEY** (ключ с https://platform.openai.com/api-keys):

```bash
set OPENAI_API_KEY=sk-...
go run ./cmd/server
```

Если в чате отправить сообщение **«маф»** (только это слово), ответ GPT показывается в чате и **озвучивается** (речь в браузере, русский голос).

## Голос по кодовому слову «маф» (Wispr)

В чате включена **интеграция с Wispr**: при открытии чата микрофон записывает голос чанками (~2.5 с), аудио отправляется на бэкенд и в **Wispr API** для распознавания речи.

- **Первый «маф»** — начинается запись (кнопка микрофона красная).
- **Второй «маф»** — запись останавливается, распознанный текст отправляется в чат как сообщение.

Нужна переменная окружения **WISPR_API_KEY** (ключ с https://wisprflow.ai / Wispr Flow):

```bash
set WISPR_API_KEY=ваш_ключ
go run ./cmd/server
```

Эндпоинт бэкенда: `POST /api/transcribe` — тело `{"audio": "base64..."}`, ответ `{"text": "..."}`.
